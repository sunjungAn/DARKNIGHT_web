    오시는 길

    상명대학교 오시는 길을 정확하게 안내해 드립니다.

    서울캠퍼스 오시는 방법 - 자가운전

    서울시 종로구 홍지문 2길 20 (우 : 03016)

    광화문/경복궁
  • 세종로에서 독립문 방면으로 좌회전 -> 경복궁역사거리에서 효자동 방면으로 우회전 후
  • 자하문터널 방면으로 직진 -> 자하문터널 통과 후 세검정삼거리에서 홍제동‧홍은사거리 방면 좌회전 ->
  • 40m 직진 후 상명대 방면으로 우회전 -> 상명대학교
    • 내부순환도로(자유로/성산대교)
  • 내부순환도로 '홍제'에서 홍은사거리 방면으로 진출 -> 홍은사거리 방면으로 직진 -> 홍은사거리에서 북악터널 방면으로 직진
  • -> 세검정삼거리에서 상명대 방면으로 좌회전 -> 상명대학교
    • 내부순환도로(북부간선/성수대교)
  • 내부순환도로 '국민대입구'에서 북악터널, 국민대입구 방면으로 오른쪽방향 진출 -> 북악터널
  • 방면으로 직진 후 통과 ->
  • 세검정삼거리까지 직진 후 상명대 방면으로 우회전 -> 상명대학교
    • 불광동/역촌동
  • 불광역사거리에서 구기터널방면으로 직진 -> 구기터널 통과 후 신영동삼거리 방면으로 직진 ->
  • 신영동삼거리에서 세검정 방면으로 우회전 -> 세검정삼거리에서 상명대 방면으로 우회전 -> 상명대학교

    • 서울캠퍼스 오시는 방법 - 대중교통

    • 서울역 → 상명대학교
    • 서울역버스환승센터에서 버스 탑승
    • 강남고속버스터미널 → 상명대학교
    • 지하철 3호선(대화, 구파발) 탑승 경복궁역 하차 후 3번 출구에서 버스 탑승
    • 김포공항 → 상명대학교
    • 지하철 5호선(마천, 상일) 탑승 광화문역 하차 후 2번 출구 KT광화문지사에서 버스 탑승
